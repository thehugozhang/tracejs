Client-side Deployment Bug Tracing Simplified

TraceJS combines custom callbacks, DOM tracking, and a centralized dashboard, empowering developers to discover, replicate, and fix deployment bugs in record time

Easily add our debugging capabilities to any JS-based project in only a few lines of code.

var data = {
    user_id: 'YOUR_USER_ID',

$.ajax('https://tracejs.co/api/v1/report', {
    type: 'POST',
    data: JSON.stringify(data),
    contentType: 'application/json'

    alert('Bug reported!');
}).fail(function(error) {
    alert('Oops... ' + JSON.stringify(error));
});
